# Voice Notes AI — Support

## Contact
For support or questions, please contact:
support@voicenotesai.app

## About the App
Voice Notes AI allows users to record voice notes and generate AI-powered summaries and actionable task lists.

## AI & Data Processing
Voice recordings and transcripts are processed to generate summaries using AI services.
No user accounts are required.
Data is not sold or used for advertising.

## Troubleshooting
If the app fails to generate a summary, please ensure you have an active internet connection.
